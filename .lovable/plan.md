

# خطة تعديل جلسات السرد القرآني

## ملخص التعديلات

تحويل جلسات السرد لتكون مفتوحة لجميع الطلاب (بدون ربط إلزامي بحلقة)، مع دعم معلم خارجي، وتحديد نطاق أحزاب السرد في بيانات الجلسة.

---

## التفاصيل التقنية

### 1. تعديل قاعدة البيانات (migration)

إضافة أعمدة جديدة لجدول `narration_sessions`:

```sql
ALTER TABLE narration_sessions
  ADD COLUMN external_teacher_name TEXT,
  ADD COLUMN external_teacher_phone TEXT,
  ADD COLUMN hizb_from INTEGER,
  ADD COLUMN hizb_to INTEGER;
```

### 2. تعديل `QuranNarration.tsx` — نموذج إنشاء/تعديل الجلسة

- إزالة حقل الحلقة من كونه أساسياً (يبقى اختيارياً للتصنيف فقط)
- إضافة حقول:
  - **اسم المعلم الخارجي** (اختياري)
  - **هاتف المعلم الخارجي** (اختياري)
  - **من حزب** / **إلى حزب** (لتحديد نطاق السرد في الجلسة)
- تحديث `sessionMutation` لإرسال الحقول الجديدة
- عرض بيانات المعلم الخارجي ونطاق الأحزاب في بطاقة الجلسة

### 3. تعديل `NarrationSession.tsx` — صفحة الجلسة

- **إزالة التبعية على الحلقة**: الجلسة تبدأ فارغة بدون طلاب تلقائيين
- يتم إضافة الطلاب يدوياً عبر البحث من جميع الطلاب النشطين
- عرض بيانات المعلم الخارجي ونطاق الأحزاب في رأس الصفحة
- تعديل رسالة "لا يوجد طلاب" لتوجيه المستخدم لإضافة طلاب

### 4. ملخص الملفات

| الملف | التعديل |
|-------|---------|
| Migration SQL | إضافة 4 أعمدة لـ `narration_sessions` |
| `QuranNarration.tsx` | حقول المعلم الخارجي + نطاق الأحزاب في نموذج الجلسة + عرضها في البطاقات |
| `NarrationSession.tsx` | إزالة جلب طلاب الحلقة تلقائياً + عرض بيانات المعلم والأحزاب |
