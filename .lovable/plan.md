

# لوحة مؤشرات بصرية لموديول السرد القرآني

## نظرة عامة

اضافة تبويب "لوحة المؤشرات" في صفحة تقارير السرد القرآني (`NarrationReports.tsx`) يحتوي على ثلاثة رسوم بيانية تفاعلية باستخدام مكتبة Recharts المتوفرة حاليا في المشروع.

---

## الرسوم البيانية المطلوبة

### 1. رسم بياني زمني (Line/Area Chart) - نسب الاجتياز عبر الجلسات
- المحور الافقي: تاريخ الجلسة
- المحور العمودي: نسبة الاجتياز (%)
- خط يوضح تطور نسبة الاجتياز من جلسة لاخرى
- يعتمد على بيانات `allAttempts` و `sessions` المتوفرة حاليا

### 2. رسم دائري (Pie Chart) - توزيع حالات الطلاب
- شرائح: ناجح / راسب / غائب / معلق
- يعتمد على بيانات الجلسة المختارة او البيانات الاجمالية
- الوان مميزة لكل حالة

### 3. رسم اعمدة (Bar Chart) - مقارنة اداء الحلقات
- عمود لكل حلقة يمثل نسبة الاجتياز او متوسط الدرجات
- يعتمد على `halaqatStats` المحسوبة حاليا في الصفحة

---

## التعديلات التقنية

### ملف واحد يتم تعديله: `src/pages/NarrationReports.tsx`

**التغييرات:**
1. استيراد مكونات Recharts: `LineChart, Line, PieChart, Pie, BarChart, Bar, XAxis, YAxis, Tooltip, ResponsiveContainer, Cell, CartesianGrid, Legend`
2. اضافة تبويب ثالث "لوحة المؤشرات" بجانب "تقارير الجلسة" و "تقارير المجمع"
3. حساب بيانات الرسم الزمني: لكل جلسة نحسب نسبة الاجتياز من `allAttempts`
4. حساب بيانات الرسم الدائري: تجميع الحالات (pass/fail/absent/pending) من كل المحاولات
5. استخدام `halaqatStats` الموجودة حاليا لرسم اعمدة المقارنة بين الحلقات

**لا حاجة لتعديل قاعدة البيانات** - جميع البيانات متوفرة حاليا في الاستعلامات الموجودة.

---

## تصميم الواجهة

```text
+--------------------------------------------------+
| تقارير الجلسة | تقارير المجمع | لوحة المؤشرات   |
+--------------------------------------------------+
|                                                    |
|  [--- رسم زمني: تطور نسب الاجتياز ---]            |
|  |     /\                                          |
|  |    /  \    /\                                    |
|  |   /    \  /  \                                   |
|  |  /      \/    \                                  |
|  +-------- الجلسات --------                        |
|                                                    |
|  +-------------------+  +---------------------+   |
|  | رسم دائري         |  | رسم اعمدة الحلقات   |   |
|  |  ناجح  60%        |  | ||||                 |   |
|  |  راسب  25%        |  | |||| ||||            |   |
|  |  غائب  15%        |  | |||| |||| ||||       |   |
|  +-------------------+  +---------------------+   |
+--------------------------------------------------+
```

- الرسم الزمني يأخذ العرض الكامل
- الرسم الدائري ورسم الاعمدة جنبا الى جنب (على الشاشات الكبيرة)
- تصميم متجاوب مع الشاشات الصغيرة
