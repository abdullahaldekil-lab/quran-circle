

# خطة تنفيذ المرحلة الثانية: التحسينات المهمة (البنود 5-9)

---

## البند 5: نظام الشارات — إصلاح التعديل والحذف + سجل تاريخي

### المشكلة
- جدول `student_badges` لا يملك سياسات RLS للتعديل (UPDATE) أو الحذف (DELETE)
- لا توجد أزرار تعديل/حذف في واجهة الشارات الممنوحة

### الحل
1. **Migration**: إضافة سياسات RLS للسماح بـ UPDATE و DELETE للمدير والمعلم، وإضافة عمود `source_detail` إذا لم يكن موجوداً
2. **تعديل `Rewards.tsx`**: إضافة أزرار حذف وتعديل الملاحظة لكل شارة ممنوحة في قسم "آخر الشارات الممنوحة"، مع إظهار تاريخ المنح ومن منحها

---

## البند 6: الإضافات المالية — إصلاح العرض + تقرير مالي

### المشكلة
- المعاملات المالية قد لا تظهر إذا لم يكن هناك حساب مالي (`financial_accounts`) مُنشأ
- لا يتم إنشاء حساب تلقائياً

### الحل
1. **تعديل `Finance.tsx`**: إذا لم يوجد حساب مالي، يتم إنشاء حساب افتراضي تلقائياً عند أول دخول للمدير
2. إضافة زر **تصدير PDF** للتقرير الشهري والتصنيفي باستخدام `jspdf` + `jspdf-autotable` (مثبتين مسبقاً)

## البند 7: نظام الباص — رقم ولي الأمر + اسم الطالب + تصدير PDF

### الحل
1. **تعديل `Buses.tsx`**:
   - تعديل استعلام `student_bus_assignments` ليجلب `students(full_name, guardian_name, guardian_phone)` بدلاً من `students(full_name, halaqa_id)` فقط
   - عرض رقم ولي الأمر بجانب اسم الطالب في جدول كل باص
   - إضافة زر **تصدير PDF** لكل باص يُنشئ ملف PDF بقائمة الطلاب وأرقام أولياء الأمور واسم السائق ورقمه

## البند 8: إدارة الحلقات — حلقة بدون مساعد + تحسين الواجهة

### الحل
1. **تعديل `Halaqat.tsx`**:
   - إضافة خيار "بدون مساعد" في قائمة المعلم المساعد (Select) عند الإضافة والتعديل
   - عند اختيار "بدون مساعد" يتم تعيين `assistant_teacher_id = null`
   - تحسين بطاقة الحلقة: إخفاء سطر "المعلم المساعد: غير محدد" إذا لم يكن هناك مساعد

## البند 9: صفحة الطلاب غير النشطين

### الحل
1. **إنشاء `src/pages/InactiveStudents.tsx`**:
   - استعلام الطلاب بحالة `withdrawn` أو `transferred` أو `inactive`
   - عرضهم في جدول مع فلتر بالحالة والبحث بالاسم
   - زر **إعادة تفعيل** يُعيد الحالة إلى `active`
   - زر **أرشفة** يُعيّن الحالة إلى `archived`
2. **تعديل `App.tsx`**: إضافة مسار `/inactive-students`
3. **تعديل `AppLayout.tsx`**: إضافة رابط "الطلاب غير النشطين" في التنقل الجانبي
4. **Migration**: إضافة قيمة `archived` إلى عمود `status` إذا لم تكن موجودة
