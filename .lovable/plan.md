

# خطة بناء صفحات نتائج الاختبار ومقارنة الحلقات

## الهيكل العام

سيتم إنشاء صفحتين جديدتين:
1. **صفحة نتائج الاختبار** (`/quiz-results`) - عرض النتائج الفردية والجماعية
2. **صفحة مقارنة الحلقات** (`/quiz-comparison`) - مقارنة أداء الحلقات

---

## المرحلة 1: صفحة نتائج الاختبار

### ملف جديد: `src/pages/QuizResults.tsx`

**القسم الأول - نتائج الطالب الفردية:**
- فلتر اختيار الحلقة ثم الطالب (نفس نمط StudentQuiz)
- بطاقة الطالب: الاسم، الحلقة، المقرر، الدرجة، التصنيف، المصحح، التاريخ
- **Gauge Chart** دائري باستخدام `PieChart` من Recharts (نصف دائرة) لعرض نسبة الدرجة
- **Line Chart** يعرض تطور درجات الطالب عبر اختباراته السابقة
- ترتيب الطالب ضمن حلقته (مثال: "الترتيب: 3 من 15") محسوب من متوسط درجات طلاب الحلقة

**القسم الثاني - نتائج الحلقة:**
- جدول طلاب الحلقة مرتب تنازلياً بالدرجات (اسم، درجة، تصنيف، تاريخ)
- متوسط درجات الحلقة مع مقارنة بالمتوسط العام
- **Donut Chart** لتوزيع التصنيفات (ممتاز / جيد جداً / جيد / يحتاج مراجعة)
- فلتر بالتاريخ (أسبوع/شهر/فصل) ومستوى الصعوبة

**مصدر البيانات:** جدول `student_quizzes` مع join على `students`, `halaqat`, `profiles` (للمصحح)

---

## المرحلة 2: صفحة مقارنة الحلقات

### ملف جديد: `src/pages/QuizComparison.tsx`

**جدول المقارنة الرئيسي:**
- أعمدة: اسم الحلقة، المعلم، عدد المختبَرين، متوسط الدرجات، نسبة الناجحين (>60%)، أعلى درجة، التصنيف العام
- ترتيب تنازلي مع ميداليات للأفضل 3

**الرسوم البيانية:**
- **Bar Chart أفقي**: متوسط درجات كل حلقة
- **Line Chart**: تطور أداء الحلقات شهرياً
- **Heatmap**: أداء كل حلقة حسب مستوى الصعوبة (جدول ملون بدرجات اللون)

**الفلاتر:**
- زمني: أسبوع / شهر / فصل / سنة
- مستوى الصعوبة
- المقرر

**التصدير:**
- PDF باستخدام `jspdf` + `jspdf-autotable` (مثبتتان)
- Excel باستخدام `xlsx-js-style` (مثبتة)

## التعديلات على الملفات الموجودة

## التفاصيل التقنية

- لا حاجة لتعديل قاعدة البيانات (البيانات موجودة في `student_quizzes` و `quiz_questions`)
- الاستعلامات تستخدم Supabase joins: `student_quizzes` → `students(full_name)`, `halaqat(name, teacher_id)`, `profiles(full_name)`
- ألوان النظام: الأخضر الداكن (`primary`) والبيج (`muted`)
- جميع المكونات تستخدم Recharts المثبتة حالياً
