

# خطة: إضافة خطاب قبول رسمي قابل للطباعة

## الفكرة
عند قبول طالب جديد من صفحة طلبات الالتحاق، يظهر زر "طباعة خطاب القبول" يفتح خطاباً رسمياً مُنسّقاً يتضمن بيانات الطالب والحلقة المعيّنة وتاريخ القبول بالهجري.

## التعديلات المطلوبة

### 1. إنشاء مكون خطاب القبول
**ملف جديد:** `src/components/enrollment/AcceptanceLetterTemplate.tsx`

- خطاب رسمي بتنسيق A4 يحتوي على:
  - شعار المجمع والعنوان الرسمي
  - رقم الخطاب (تلقائي من ID الطلب)
  - التاريخ الهجري
  - "نفيدكم بقبول ابنكم [اسم الطالب] في مجمع حويلان لتحفيظ القرآن الكريم"
  - بيانات الطالب: الاسم، الحلقة المعينة، اسم ولي الأمر
  - تعليمات وملاحظات عامة للالتحاق
  - توقيع مدير المجمع وختم
- يستخدم نفس آلية الطباعة الموجودة في `EnrollmentPrintTemplate` (window.open + print)
- التواريخ بالهجري عبر `formatDateSmart`

### 2. إضافة زر الطباعة في صفحة طلبات الالتحاق
**ملف:** `src/pages/EnrollmentRequests.tsx`

- إضافة زر "طباعة خطاب القبول" (أيقونة Printer) بجوار الطلبات المقبولة (`status === "approved"`)
- الزر يفتح dialog يعرض معاينة مبسطة + زر طباعة
- يمرر بيانات الطلب واسم الحلقة المعينة للمكون

## التفاصيل التقنية
- لا حاجة لتعديلات في قاعدة البيانات
- المكون يستقبل: اسم الطالب، اسم ولي الأمر، رقم الهاتف، اسم الحلقة، تاريخ القبول
- نفس أسلوب التصميم المستخدم في `EnrollmentPrintTemplate` (ألوان ذهبية + أزرق داكن)
